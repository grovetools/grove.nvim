The following table provides a reference for the default keybindings available in `grove-nvim`.

| Keybinding | Mode | Description |
| :--- | :--- | :--- |
| `<leader>fp` | Normal | Open the Grove Plan picker UI. |
| `<leader>fpx` | Normal | Create a new Plan by extracting content from the current buffer. |
| `<leader>fc` | Normal | Run a Grove chat session on the current buffer. |
| `<leader>jn` | Normal | Add a new job to the active plan using a form. |
| `<leader>ji` | Normal | Add a new job to the active plan using the terminal UI. |
| `<leader>fq` | Visual | Append the visual selection to the target file and prompt for a question. |
| `<leader>fr` | Visual | Append the visual selection, ask a question, and immediately run a chat on the target file. |
